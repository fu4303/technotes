# Regular Expression

ziishaned/learn-regex (opens new window)

  • Syntax : /pattern/modifiers

# Modifiers

  • i - case-insensitive
  • g - global (All Not just first match)
  • m - multiline

# Patterns

  • [abc] - chars
  • [a-z] - all
  • [A-Z] - all uppercase
  • [0-9] - digits
  • [x|y] - x or y

# Metacharacters

  • \d - digits
  • \s - whitespace
  • \uxxxx - unicode char hex value

# Quantifiers

  • +n / n+ - Any match ( atleast 1 matched case)
  • *n / n* - Any match ( 0 - n matched case)
  • ?n / n? - Any match ( 0 - 1 matched case)
