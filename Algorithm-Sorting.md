#### # BUBBLE SORT

- Sorts elements list like an *array*.
- Compares 2 adjacent elements
- eg :
    > Take [3,1,5,2]
    >
    > Iteration 1 :
        step 1 -> [1,3,5,2]         // swap
             2 -> [1,3,5,2]         // no
             3 -> [1,3,2,5]         // swap
    > Iteration 2 :
        step 1 -> [1,3,2,5]         // no
             2 -> [1,2,3,5]         // swap
             3 -> [1,2,3,5]         // no
    > Iteration 3 : 
        step 1 -> [1,2,3,5]         // no
             2 -> [1,2,3,5]         // no
             3 -> [1,2,3,5]         // no (SO STOP ITERATION)
